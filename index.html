HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.
Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance he document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.

br과 p의 차이?

br은 단순 줄바꾸기로, 사용자의 입맛에 따라 줄바꿈을 자유롭게 할 수 있다는 장점이 있지만, 문장에 의미를 부여하지 않는다
하지만 p는 단락으로서 정보의 단위를 묶어준다는 장점을 가지고 있다. 그러나, 디자인의 자유도에서는 br보다 떨어진다. 이를 보완하기 위해 CSS태그를 활용할 수 있다. 아래는 그 예시이다.

Tag의 중요성

태그(Tag)는 왜 중요할까?
그 이유는 Tag로 텍스트의 정보를 나타낼 수 있기 때문이다. 가령 예를 들자면 '제목'을 표기하는 태그인 heading tag (h1~h6)을 사용할때, 검색 엔진은 해당 태그를 '제목'으로 인식하고 선별적으로 수집할 수 있다.
하지만, 디자인이 같고 heading tag를 사용하지 않았을 경우에는 그 정보가 '제목'인지 프로그램상으로 알 수 없기 때문에 정보의 노출도가 달라진다.
